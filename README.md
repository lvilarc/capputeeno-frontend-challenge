## üñ•Ô∏è Installation

### Frontend

1. Clone this repo
```bash
git clone https://github.com/lvilarc/capputeeno-frontend-challenge.git
```

2. Go to the project folder
```bash
cd capputeeno-frontend-challenge
cd capputeeno
```

3. Install dependencies
```bash
npm install
```

3. Run Frontend
```bash
npm run dev
```

### Running the API


2. Go to the API folder

```bash
cd capputeeno-frontend-challenge
cd api
```

3. Install dependencies
```bash
npm install
```

3. Run start
```bash
npm start
```
